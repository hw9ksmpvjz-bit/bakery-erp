### 强制开发流程（永久遵守）
```
1. 编写代码 → 2. 本地构建测试(npm run build) → 3. 修复所有错误 → 4. 提交到GitHub → 5. 同步到Codespaces → 6. 自行打开测试链接验证 → 7. 确认正常后通知用户验收
```

**绝对禁止**：
- ❌ 提交未经验证的代码
- ❌ 未本地构建测试就提交
- ❌ 未自行验证测试链接就通知用户
