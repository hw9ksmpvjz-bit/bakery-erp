## 二、導航結構對照

### 優化前
```
頂部導航: 儀表板 | 商品 | 採購 | 生產 | 銷售 | 庫存 | 財務 | 報表 | 設置
儀表板內: 4個重複菜單按鈕（商品/銷售/庫存/設置）
```

### 優化後（參照金蝶）
```
頂部導航:
├── 基礎資料 [下拉] → 商品管理、供應商管理
├── 採購管理
├── 生產管理
├── 銷售管理 [下拉] → POS開單、會員管理、門店管理
├── 庫存管理
├── 財務管理 [下拉] → 財務報表、應收應付
└── 系統設置

儀表板內: 6個功能模塊卡片（無重複入口）
```
